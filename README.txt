Cara menjalankan:
1. Buka folder ini di VS Code.
2. Klik kanan index.html -> Open with Live Server.
3. Tes API dari halaman utama.
4. Buat data pengguna Admin/Petugas lewat spreadsheet/API bila belum ada.
5. Generate QR/token dari halaman admin.
6. Saat dipindah ke GitHub Pages, ubah FRONTEND_BASE_URL pada Code.gs.

Catatan:
- QR localhost tidak dapat dibuka dari HP lain.
- Akses token frontend belum sama dengan autentikasi keamanan penuh.
